# Contributing to Awesome LLM Knowledge Systems

Welcome! This is a community-maintained list of tools, papers, frameworks, and resources related to LLM knowledge systems. We appreciate your contributions.

## How to Contribute

1. **Fork** this repository
2. **Edit** the relevant file (usually `README.md`)
3. **Submit a Pull Request**

That's it!

## Submission Requirements

Each entry you add must include:

- **Name** of the tool, paper, or resource
- **URL** linking to the official source
- **One-line description** explaining what it does or why it matters

### Scope

Submissions must be related to LLM knowledge systems, including but not limited to:

- RAG (Retrieval-Augmented Generation) frameworks and pipelines
- Vector databases and embedding models
- Knowledge graphs and graph-based retrieval
- Agent memory systems (short-term, long-term, episodic)
- Context engineering and context window management
- Harness engineering and agent orchestration
- MCP (Model Context Protocol) tools and servers
- Skill systems and progressive disclosure
- Long-context models and optimization techniques

### Quality Bar

- **Tools and frameworks** must be actively maintained (commits within the last 6 months)
- **Papers** should be highly cited or from reputable venues (top conferences, well-known research labs)
- **No self-promotion spam** -- if you are submitting your own project, it must genuinely fit the scope and meet the quality bar above
- **Links must be working** -- broken links will not be accepted

### Formatting

- Entries must be placed in **alphabetical order** within their section
- Follow the existing format used in that section
- One entry per pull request is preferred (makes review faster)

## Pull Request Format

- PR title should follow the format: **"Add [Tool/Paper Name]"**
- Include a brief note in the PR description explaining why this resource belongs in the list
- Use the PR template provided -- it will auto-populate when you open a new PR

## Reporting Issues

- **Broken link?** Open an issue using the "Report Broken Link" template
- **Want to suggest a resource but not sure?** Open an issue using the "Suggest a Resource" template

## Code of Conduct

Be respectful. Keep discussions constructive. We are all here to build a better knowledge base for the community.

Thank you for contributing!
